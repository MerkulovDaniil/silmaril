# Welcome to Silmaril

**Your vault stays in one place. You access it from anywhere.**

Silmaril is a self-hosted, mobile-first web UI for [Obsidian](https://obsidian.md) vaults. Single Python file. Zero config. Install and go.

## Try it right now

This entire site **is** Silmaril. Everything you see is rendered from plain markdown files.

> [!tip] This is a live playground
> You can edit any page, change icons, search — go ahead and break things. Every 15 minutes this site rebuilds from the repository.

## Quick start

```bash
pip install silmaril
silmaril --vault /path/to/your/vault
```

Open `http://localhost:8000` — done.

## Why Silmaril?

| Problem | Silmaril |
|---------|----------|
| Obsidian Sync costs money | Free, self-hosted |
| Sync plugins break with conflicts | No sync — one vault, access from anywhere |
| Can't read notes on phone without the app | Any browser works |
| Complex setup with databases and configs | `pip install` and one command |
